FAQ

Questions? Answers.

Everything you need to know before joining.

What exactly is a "skill"?

A skill is a detailed automation playbook that your AI agent (like Claude, GPT, or any OpenClaw-compatible agent) reads and follows step by step. It tells the AI exactly how to perform a complex task -- like producing a video ad -- including which APIs to call, what tools to use, how to handle errors, and when to ask you for approval. Think of it as hiring a specialist, except the specialist is your AI and it works 24/7.

Do I need coding experience?

No. The skills are written so that AI agents can follow them, not you. You provide inputs (like an audio file and an image), approve outputs along the way, and get a finished product. The AI handles all the technical execution -- ffmpeg commands, API calls, image processing, everything.

What AI agent do I need?

Dope Skills are designed for OpenClaw agents, but the skill files are standard markdown and can be adapted for Claude Code, Codex, or any agent that reads instruction files. The setup guide covers everything you need to get started.

Are there additional API costs?

Some skills use external APIs (like Creatify for avatar generation). These have their own costs, which are documented in each skill's setup guide. Many skills use free-tier APIs or run locally with zero cost. The Talking Head B-Roll Video skill, for example, uses Creatify (~$5-10 per video), Gemini (free), and Grok (free).

How often do new skills drop?

We're actively building new skills based on real production needs and member requests. Expect 1-2 new skills per month. As a member, you get every new skill automatically -- no extra charge.

Can I cancel anytime?

Monthly members can cancel anytime. Annual members are locked in for the year but get a massive discount and founding member perks that won't be available later.

What if I need a custom skill?

Annual members can submit custom skill requests. If it's something that benefits the community, we'll build it and add it to the library. If it's highly specific to your business, we can discuss custom development.
